Robot.txt

Robots.txt is a text file in hte top level of the heirarchy and is designed to act as an indicator which pages a "robot" or search engine robot should disregard. This text file contains keywords and paths to the forbidden html pages

Humans.txt

Humans.txt is a file that is similar to "robot.txt" and contains the information about the people behind a page, when it was last updated, their contact information or copyright/trademarks regarding usage and information presented on the site.

Open Graph

Open graph is a protocol used on webpages to enable them to become graph objects and thus share functionality as other related objects. This is achieved by adding basic meta data in the headers of your web pages that are used to relate to your web page

Why we separate HTML and CSS

The reason for separating HTML and CSS is because it makes the code and content much easier to read and understand whereas if both CSS and HTML were all combined into one along with javascript, it would be very messy and hard to find things. It also limits code duplication and is because of that a more efficient way since you only have to specify things once and then reference them via a tag or it may even be automatic. It also reduces bandwidth usage which is important on lower bandwidth modems or networks such as 56k etc. It also allows for changing layout and style without touching the page content at all.

Media Queries

A Media Query is a CSS technique which was introduced in CSS3. It uses a rule; "@media" to include/invoke a block of css properties if a certain condition is true. This condition could for example be screen size/resize or say, if the OS is android or iOS.

Block or Inline Elements

A block level element has a rule to always start on a new line and use the full available width which means that it stretches itself as far right and left as it can.
An Inline Element however does not start on a new line and only takes up as much space as needed.
